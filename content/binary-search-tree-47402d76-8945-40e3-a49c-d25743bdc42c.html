Binary Search Tree Algorithm

Description

Binary Search tree is a binary tree in which each internal node x stores an element such that the element stored in the left sub tree of x are less than or equal to x and elements stored in the right sub tree of x are greater than or equal to x. This is called binary-search-tree property.
